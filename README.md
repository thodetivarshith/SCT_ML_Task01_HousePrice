# House Price Prediction using Linear Regression

## Overview
This project predicts house prices using a Linear Regression model trained on the Kaggle House Prices dataset.

## Dataset
House Prices - Advanced Regression Techniques

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Joblib

## Workflow
1. Data Loading
2. Exploratory Data Analysis
3. Missing Value Handling
4. Feature Selection
5. Train-Test Split
6. Linear Regression Model Training
7. Model Evaluation
8. Feature Importance Analysis
9. Model Saving

## Evaluation Metrics

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

## Conclusion

The Linear Regression model successfully predicts house prices based on housing characteristics and provides a strong baseline for regression tasks.
